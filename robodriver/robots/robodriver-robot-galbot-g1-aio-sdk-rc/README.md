# Robodriver-Robot-Galbot-G1-AIO-SDK

在开始前，请确保您已经完成 [RoboDriver文档/概览/安装与部署](https://flagopen.github.io/RoboDriver-Doc/docs/overview/installation/) 中的步骤或已完成RoboDriver仓库README中的 `快速入门` 。

## 环境部署

新建一个终端，且暂时不激活任何环境。

确保进入RoboDriver目录，如果已经进入就跳过：

```bash
cd RoboDriver/
```

激活 `RoboDriver` 环境：

```bash
source .venv/bin/activate
```

进入到 `robodriver-robot-galbot-g1-aio-sdk-rc` 目录。

```bash
cd robodriver/robots/robodriver-robot-galbot-g1-aio-sdk-rc
```

安装依赖

```bash
uv pip install -e .
```

## 启动

新建终端并进入 `RoboDriver` 项目目录，如果已经进入就跳过：

```bash
cd RoboDriver/
```

激活 `RoboDriver` 环境：

```bash
source .venv/bin/activate
```

运行：

```bash
robodriver-run --robot.type=galbot_g1_aio_sdk_rc
```
